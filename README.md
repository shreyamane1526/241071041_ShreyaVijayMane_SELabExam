# Smart Waste System

This project is a Smart Waste Management System developed for the Software Engineering Lab Exam.

## How to Run

1. Make sure you have Python 3 installed.
2. Install required libraries (if any).
3. Run the main script:
   ```sh
   python3 smart_waste_system-1.py
   ```

## Files

- `smart_waste_system-1.py` — Main Python script
- `BlackBox_TestCases-1.xlsx` — Black box test cases
- `WhiteBox_TestCases-1.xlsx` — White box test cases
